# DebtCrack 💳

**Free debt payoff calculator — [debtcrack.pro](https://debtcrack.pro)**

Calculate your debt-free date, compare snowball vs avalanche methods and see exactly how much interest you can save.

---

## Features

- 💳 **Credit card payoff calculator** — balance, APR, monthly payment → exact debt-free date
- ❄️ **Debt snowball calculator** — pay smallest debt first for quick wins
- 🏔️ **Debt avalanche calculator** — pay highest interest first to save the most money
- 📅 **Debt-free date** — exact month and year you'll be debt free
- 💰 **Interest savings** — see how much you save vs minimum payments
